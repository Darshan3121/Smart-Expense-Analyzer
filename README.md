# Smart Expense Analyzer & Budget Alert System (Python)

A real-world Python application that helps users track expenses, analyze spending patterns, visualize data, and receive budget alerts.

## Features
- Add and store expenses persistently using CSV
- View all expenses and total spending
- Category-wise expense analysis using pandas
- Budget alert system with warnings
- Visual charts (bar & pie) using matplotlib
- Menu-driven, user-friendly interface

## Tech Stack
- Python
- Pandas
- Matplotlib
- CSV file handling

## How to Run

pip install -r requirements.txt  
python expense_tracker.py
